# AI Inbound Receptionist System

## Overview
This project is a production-ready AI voice receptionist that answers inbound calls and automates appointment scheduling, CRM updates, and confirmation emails.

---

## Features
- Answers inbound calls 24/7
- Books appointments automatically
- Reschedules appointments
- Cancels appointments
- Creates calendar events
- Updates CRM in real time
- Sends confirmation emails

---

## Tech Stack
- Voice AI
- n8n automation
- Calendar API integration
- CRM API integration
- Email automation
